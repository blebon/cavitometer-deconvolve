Introduction¶

Cavitometer-deconvolve is a Python library that converts raw hydrophone voltage signals to pressure.

If you have any question or find a bug, open an issue.

Download and install¶

This package is in the Python Package Index, so pip install cavitometer-deconvolve is enough. You can also clone the package on Github.
